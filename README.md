<h1 align="center">Java Console Game</h1>

This is a simple Stone Paper Scissor game built using Java. It is a console-based application where the user plays against the computer.

The computer randomly selects Stone, Paper, or Scissor, and the winner is decided based on the standard game rules.

 <h2>Features</h2>

* Console-based interaction

* Random computer choice

* Instant result (Win / Lose / Draw)

* Simple and easy to understand code
